# 🌍 EcoTrack — Carbon Footprint Tracking Platform

A production-ready full-stack application for tracking and reducing carbon footprints.

## Tech Stack

- **Frontend:** React.js (Vite)
- **Backend:** Node.js + Express
- **Database:** PostgreSQL

## Getting Started

```bash
# Install all dependencies
npm run install:all

# Start development servers
npm run dev
```

## Project Structure

```
EcoTrack/
├── client/          # React.js frontend application
├── server/          # Node.js + Express backend API
├── database/        # PostgreSQL migrations, seeds & scripts
├── docs/            # Project documentation & API specs
├── scripts/         # DevOps, CI/CD & automation scripts
└── .github/         # GitHub Actions workflows
```
